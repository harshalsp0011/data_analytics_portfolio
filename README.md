

================================================================================
                            DATA ANALYTICS PORTFOLIO
================================================================================

PROJECT OVERVIEW
----------------
This project is a personal portfolio website designed to showcase interactive 
Business Intelligence (BI) dashboards and data analytics projects. It serves as a 
central hub to present Power BI reports, professional experience, and technical 
skills in a clean, responsive interface.

The site is built as a single-page application (SPA) using HTML, JavaScript, and 
Tailwind CSS, allowing for dynamic content rendering without page reloads.

KEY FEATURES
------------
* **Project Gallery:** A home view that lists available dashboard projects with 
    descriptions and tool tags (e.g., Power BI, DAX).
* **Interactive Embeds:** Full-screen capable containers for embedding Power BI 
    reports directly into the browser.
* **Responsive Design:** Styled with Tailwind CSS to work on mobile and desktop 
    devices.
* **Contact Section:** A built-in functional form integrated with Formspree for 
    direct messaging, alongside links to social profiles.

INCLUDED PROJECTS
-----------------
1.  **KPI Performance Report**
    * Description: Interactive analysis of orders, revenue, and profit trends.
    * Tools: Power BI, DAX, Data Modeling, QA Profiling.

2.  **Global Sales Overview (Placeholder)**
    * Description: A reserved slot for future dashboards (e.g., Sales, HR, or 
        Marketing analytics).

TECHNICAL STACK
---------------
* **Frontend:** HTML5, JavaScript (Vanilla)
* **Styling:** Tailwind CSS (loaded via CDN)
* **Visualization:** Microsoft Power BI (Embedded via iframe)
* **Icons:** Heroicons (SVG)

HOW TO RUN
----------
1.  Ensure you have an active internet connection (required to load the Tailwind 
    CSS library and Power BI reports).
2.  Simply open the `index.html` file in any modern web browser (Chrome, Edge, 
    Firefox, Safari).

CONFIGURATION & CUSTOMIZATION
-----------------------------
To customize this portfolio for your own use, edit the `index.html` file:

1.  **Adding Projects:**
    Locate the `const projects` array inside the `<script>` tag. Add new objects 
    following the existing structure to display new dashboards.

2.  **Contact Form:**
    Locate the `<form>` tag in the Contact Section. Replace the `action` URL with 
    your own Formspree endpoint ID.

3.  **Personal Information:**
    Update the email, resume link, and social media URLs (LinkedIn/GitHub) in the 
    HTML body.
